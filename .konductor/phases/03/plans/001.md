# Plan 01: Self-Improvement Loop Script

## Goal
Build the autonomous improvement loop that analyzes benchmarks, improves skills, validates improvements, and reverts regressions.

## Tasks

### Task 1: Create improvement loop script

- **files:** `tests/e2e/improve.sh`
- **action:** Create script that: (1) sources lib.sh, (2) runs initial benchmark via benchmark.sh, (3) for each iteration (max 3): calls kiro-cli chat to analyze last-results.toml and suggest improvements to a specific skill, saves current skills via git stash, applies the improvement, re-runs benchmark, runs compare.sh against baseline — if regression detected, reverts via git stash pop; if improved, updates baseline with new values, (4) prints summary of iterations and net improvement.
- **verify:** `bash -n tests/e2e/improve.sh` passes
- **done:** Improvement loop runs locally

### Task 2: Add analysis prompt and revert logic

- **files:** `tests/e2e/improve.sh`
- **action:** Write the analysis prompt that tells the agent: "Read tests/e2e/last-results.toml. Identify the worst-performing step. Read the corresponding skill file in skills/konductor-{step}/SKILL.md. Generate an improved version that addresses the performance issue. Write the improved version to the same path. Only modify the SKILL.md file, nothing else." Add git-based revert: `git stash push -m "pre-improve-N" -- skills/` before each change, `git stash pop` on regression.
- **verify:** `bash -n tests/e2e/improve.sh` passes
- **done:** Analysis and revert logic complete
