# Plan 02: CI Workflow

## Goal
Create a GitHub Actions workflow that runs all test tiers and optionally the benchmark + improvement loop.

## Tasks

### Task 1: Create test workflow

- **files:** `.github/workflows/test.yml`
- **action:** Create workflow triggered on pull_request. Jobs: (1) `test` — checkout, install Rust, cargo test (runs unit + MCP integration tests), (2) `e2e` — conditional on KONDUCTOR_E2E secret, runs benchmark.sh, uploads results.toml as artifact, runs compare.sh to check for regressions. The e2e job needs kiro-cli installed and authenticated.
- **verify:** Workflow YAML is valid
- **done:** CI workflow covers all test tiers

### Task 2: Add benchmark artifact upload and regression check

- **files:** `.github/workflows/test.yml`
- **action:** In the e2e job, after benchmark.sh completes, upload last-results.toml as a workflow artifact. Run compare.sh — if it exits 1 (regression), fail the job. Add a step that posts a summary comment with key metrics.
- **verify:** Workflow YAML has upload-artifact and compare steps
- **done:** CI reports benchmark results and catches regressions
