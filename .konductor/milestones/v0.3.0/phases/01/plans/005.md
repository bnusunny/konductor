# Plan 05: MCP Error Handling and CLI Polish

## Goal
Standardize error responses across all MCP tool handlers and add CLI polish (version flag). This plan depends on plans 1, 3, and 4 because it touches the same tool handlers after they've been updated.

## Dependencies

This plan depends on:
- Plan 1: State machine hardening (state tools may have changed)
- Plan 3: Frontmatter parser rewrite (plans_list tool changed)
- Plan 4: Config module (new config_get tool added)

## Tasks

### Task 1: Define structured error format and helper

- **files:** `konductor-cli/src/mcp.rs`
- **action:** Create a `tool_error(code, message)` helper function that returns a JSON string `{"error": true, "code": "...", "message": "..."}`. Define error codes: `STATE_NOT_FOUND`, `INVALID_TRANSITION`, `PARSE_ERROR`, `IO_ERROR`, `NO_PLANS`, `INVALID_CONFIG`.
- **verify:** `cargo check` passes
- **done:** Error helper function exists

### Task 2: Apply structured errors to all tool handlers

- **files:** `konductor-cli/src/mcp.rs`
- **action:** Update `state_get`, `state_transition`, `state_add_blocker`, `state_resolve_blocker`, `plans_list`, `status`, and `config_get` to use `tool_error()` for all error paths. Ensure every `Err(e)` branch returns structured JSON.
- **verify:** `cargo test` passes, error responses are valid JSON
- **done:** All tool error responses are structured JSON

### Task 3: Add --version flag

- **files:** `konductor-cli/src/main.rs`, `konductor-cli/Cargo.toml`
- **action:** Add `#[command(version)]` to the Clap `Cli` struct so `konductor --version` prints the version from Cargo.toml.
- **verify:** `cargo run -- --version` prints version string
- **done:** Version flag works

### Task 4: Final clippy and test pass

- **files:** All Rust source files
- **action:** Run `cargo clippy -- -D warnings` and fix any warnings. Run `cargo test` and ensure all tests pass. Run `cargo fmt --check` and fix formatting.
- **verify:** `cargo clippy -- -D warnings` exits 0, `cargo test` exits 0
- **done:** Zero clippy warnings, all tests pass
