# Plan 01: MCP Test Harness and Tool Tests

## Goal
Build the integration test harness using rmcp client and test all 7 MCP tools against the real `konductor mcp` binary.

## Tasks

### Task 1: Add dev-dependencies for integration testing

- **files:** `konductor-cli/Cargo.toml`
- **action:** Add `rmcp = { version = "1.2", features = ["client", "transport-child-process"] }` and `anyhow = "1"` to `[dev-dependencies]`.
- **verify:** `cargo check --tests` passes
- **done:** Client dependencies available for integration tests

### Task 2: Create test harness helper and state_get/config_get tests

- **files:** `konductor-cli/tests/mcp_tools.rs`
- **action:** Create integration test file. Write a `setup_test(state_toml: &str)` helper that: creates a temp dir, writes `.konductor/state.toml` and `.konductor/config.toml`, builds the binary path from `env!("CARGO_BIN_EXE_konductor")`, spawns `konductor mcp` with `current_dir` set to temp dir, connects rmcp client, returns `(client, temp_dir)`. Write tests for `state_get` (valid state, missing state) and `config_get` (returns defaults).
- **verify:** `cargo test --test mcp_tools` passes
- **done:** Test harness works, state_get and config_get tested

### Task 3: Write state_transition and blocker tests

- **files:** `konductor-cli/tests/mcp_tools.rs`
- **action:** Add tests for: `state_transition` valid transition (initialized→planned), `state_transition` invalid transition (initialized→executed returns error), `state_add_blocker` adds blocker and sets status, `state_resolve_blocker` resolves and clears status. Each test uses fresh temp dir.
- **verify:** `cargo test --test mcp_tools` passes
- **done:** Transition and blocker tools fully tested

### Task 4: Write plans_list and status tests

- **files:** `konductor-cli/tests/mcp_tools.rs`
- **action:** Add tests for: `plans_list` with plans present (create a plan file in temp dir, verify it appears in response), `plans_list` with no plans dir (returns error), `status` returns structured report with project name, phase, step, and next suggestion.
- **verify:** `cargo test --test mcp_tools` passes
- **done:** All 7 MCP tools tested
